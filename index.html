Soal 01
Konversi Suhu
Celsius → Fahrenheit (F = 9/5 × C + 32) dan Kelvin (K = C + 273.15)
Pseudocode
Algoritma Konversi Suhu
ALGORITMA KonversiSuhu DEKLARASI Celsius, Fahrenheit, Kelvin : Real DESKRIPSI Output("Masukkan suhu dalam Celsius: ") Input(Celsius) Fahrenheit ← (9/5 × Celsius) + 32 Kelvin ← Celsius + 273.15 Output("Fahrenheit = ", Fahrenheit) Output("Kelvin = ", Kelvin) SELESAI
Cara Kerja Program menerima satu nilai suhu Celsius, kemudian langsung menghitung kedua hasil konversi sekaligus menggunakan dua rumus yang berbeda, lalu menampilkan keduanya dalam satu kali jalan program.

Tahun Kabisat
Aturan: habis dibagi 400, atau habis dibagi 4 tapi tidak 100. Uji: 2000, 1900, 2024, 2023
Pseudocode
Algoritma Pengecekan Tahun Kabisat
ALGORITMA CekTahunKabisat DEKLARASI Tahun : Integer DESKRIPSI Output("Masukkan angka Tahun: ") Input(Tahun) JIKA (Tahun mod 400 = 0) MAKA Output("Tahun Kabisat") SEBALIKNYA JIKA (Tahun mod 100 = 0) MAKA Output("Bukan Tahun Kabisat") SEBALIKNYA JIKA (Tahun mod 4 = 0) MAKA Output("Tahun Kabisat") SEBALIKNYA Output("Bukan Tahun Kabisat") AKHIR JIKA SELESAI
Logika Pengambilan Keputusan Pemeriksaan dilakukan dari kondisi paling spesifik ke paling umum: pertama cek habis 400 (selalu kabisat), lalu habis 100 (bukan kabisat meskipun habis 4), terakhir habis 4 (kabisat). Urutan ini penting agar logika benar.

Soal 03
Deret Fibonacci
Menampilkan F(1) hingga F(10) menggunakan struktur perulangan dengan tabel iterasi
Pseudocode
Algoritma Deret Fibonacci F(1)–F(10)
ALGORITMA DeretFibonacci DEKLARASI a, b, c, i : Integer DESKRIPSI a ← 1 { nilai F(1) } b ← 1 { nilai F(2) } Output("F(1) = ", a) Output("F(2) = ", b) UNTUK i ← 3 SAMPAI 10 LAKUKAN c ← a + b Output("F(", i, ") = ", c) a ← b b ← c AKHIR UNTUK SELESAI
Cara Kerja Loop Variabel a dan b menyimpan dua nilai Fibonacci sebelumnya. Di setiap iterasi, c = a + b dihitung, lalu a digeser ke b dan b digeser ke c. Ini adalah teknik sliding window yang efisien karena hanya butuh tiga variabel.

Soal 04
Kalkulator BMI
BMI = Berat (kg) / Tinggi² (m) · Klasifikasi 4 kategori + Kode Python manual
Pseudocode
Algoritma Kalkulator BMI
ALGORITMA KalkulatorBMI DEKLARASI Berat : Integer Tinggi, BMI : Real TinggiM : Real { Tinggi dalam meter } FUNGSI HitungBMI(Berat, TinggiM) : Real KEMBALIKAN Berat / (TinggiM × TinggiM) PROSEDUR CekBMI(BMI) JIKA BMI < 18.5 MAKA Output("Kurus (Underweight)") SEBALIKNYA JIKA BMI < 25.0 MAKA Output("Normal (Healthy Weight)") SEBALIKNYA JIKA BMI < 30.0 MAKA Output("Gemuk (Overweight)") SEBALIKNYA Output("Obesitas (Obese)") AKHIR JIKA DESKRIPSI UTAMA Output("Masukkan Berat Badan (kg): ") Input(Berat) Output("Masukkan Tinggi Badan (cm): ") Input(Tinggi) TinggiM ← Tinggi / 100 BMI ← HitungBMI(Berat, TinggiM) Output("Nilai BMI Anda: ", BMI) CekBMI(BMI) SELESAI
Flowchart
Kode Python
kalkulator_bmi.py
# ============================================= # Kalkulator BMI – Alfin Maulana Yudistira # NIM: 250601095 | Kelas B Sore – TI # ============================================= def hitung_bmi(berat, tinggi_m): """Menghitung nilai BMI dari berat (kg) dan tinggi (meter).""" bmi = berat / (tinggi_m ** 2) return bmi def cek_kategori(bmi): """Menentukan kategori berat badan berdasarkan nilai BMI.""" if bmi < 18.5: return "Kurus (Underweight)" elif bmi < 25.0: return "Normal (Healthy Weight)" elif bmi < 30.0: return "Gemuk (Overweight)" else: return "Obesitas (Obese)" # ---- Program Utama ---- print("=" * 40) print(" KALKULATOR BODY MASS INDEX (BMI)") print("=" * 40) berat = int(input("Masukkan Berat Badan (kg) : ")) tinggi = float(input("Masukkan Tinggi Badan (cm): ")) tinggi_m = tinggi / 100 bmi = hitung_bmi(berat, tinggi_m) kategori = cek_kategori(bmi) print("-" * 40) print(f"Nilai BMI Anda : {bmi:.2f}") print(f"Kategori : {kategori}") print("=" * 40)
